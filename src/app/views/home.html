Planificador de horarios

Para cargar datos con archivos excel se necesita que los headers tengan el siguiente formato(sin importar el orden):

Header 'Código' para identificar la materia

Header 'Cuatrimestre' para identificar el cuatrimestre al que corresnde la comision

Header 'Comisión' para identificar el numero de comision

Header 'Docente' para identificar al profesor de la catedra

Header 'Dia' para identificar el dia en el que se dicta la materia(IMPORTANTE: Dia no debe llevar tilde)

Header 'Hora Inicio' para identificar el horario de inicio del dictado de la materia(Formato HH:mm)

Header 'Hora Final' para identificar el horario final del dictado de la materia(Formato HH:mm)
